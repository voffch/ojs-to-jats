Это веб-приложение для подготовки JATS XML метаданных научных публикаций на основании текста, введенного в форму. Основная задача - генерировать валидные [JATS XML v. 1.4](https://jats.nlm.nih.gov/archiving/tag-library/1.4/), пригодные для загрузки в [ИС Метафора](https://metafora.rcsi.science/). Приложение написано для современного [Firefox](https://www.firefox.com) но, скорее всего, не откажется работать и в других браузерах.

Вы запустили приложение в режиме веб-страницы. Расширенные возможности (загрузка информации из OJS и отправка в ИС Метафора по API) доступны только в режиме [расширения браузера](https://addons.mozilla.org/ru/firefox/addon/ojs-to-jats/). Веб-страница генерирует такой же файл JATS XML, как и расширение, но заполнять форму можно пока что только вручную или из имеющегося JATS XML, сгенерированного этим приложением.

Порядок действий с веб-страницей такой: 

- заполняете форму, 
- смотрите под формой метаданные, воспроизведенные из сгенерированного XML (или даже сам XML), 
- проверяете правильность введенных данных, 
- и скачиваете XML по кнопке в меню.
