### Метафора API

В данной версии приложения для простоты уникальным идентификатором статьи в ИС Метафора служит DOI. Соответственно, для работы с Метафорой вам нужно ввести ключ API (сохраняется в браузере) и DOI статьи (заполняется в форме).

После того, как вы это сделали, нажимайте кнопку "Поиск по DOI" чтобы узнать статус статьи в Метафоре. Непосредственно статус показан в текстовом виде под кнопкой. Например, для статьи, еще не загруженной в Метафору, эта часть веб-приложения после поиска будет выглядеть так:

![выгрузка в Метафору](./img/metafora-controls.png)

Для статьи, статус которой известен, по нажатию кнопок доступен ряд действий. Должно быть очевидно что они делают. Если кнопки неактивны, значит, это действие со статьей с таким статусом выполнить нельзя (например, нельзя удалить подписанную статью, или загрузить новый файл XML поверх существующего и не удаленного старого).

По нажатию кнопки после того, как в ИС Метафора отправится запрос и будет получен ответ, результаты отображаются внизу под кнопками в виде короткой словесной расшифровки и полного ответа сервера Метафоры (ниже, с подсветкой синтаксиса). В первую очередь смотрите на короткую расшифровку. На полный ответ нужно обращать внимание только если что-то пошло не так - например, XML не загрузился из-за ошибки в каком-либо из полей. Во многих случаях по диагностическому сообщению от сервера можно понять в чем именно проблема.
